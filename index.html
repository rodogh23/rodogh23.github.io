Programa para encriptar toda clase de ficheros binarios (fotos, ejecutables, peliculas, ...) y archivos de texto (passwords, notas privadas, ..)

Crenalme no guarda ninguna password dentro del fichero sino que convierte cada byte del mismo en un nuevo byte que está en función de toda la password introducida.

Puedes descargarte el codigo fuente del programa. He pasado tambien el algoritmo a JavaScript, de forma que puedas poner tus propias paginas web encriptadas. Pincha aqui para ver un ejemplo de una pagina con hiperlinks.

Aunque Carlos y Chema me dan la coña con que no hay nada indesencriptable, voy a poner unos ficheros encriptados para el que quiera o pueda les de la razon.

El primero es uno 'facilito', aunque aviso de antemano que la password tiene mas de 10 caracteres. Puedes encontrar aqui la version html del fichero encriptado. El archivo encriptado en web ocupa 40Kb y es posible que el navegador tarde en desencriptarlo. Si quieres, tambien puedes bajartelo de aqui e intentarlo de forma mas masiva.

El segundo es uno de los que es 'imposible' de desencriptar. Esta encriptado mas de una vez sobre si mismo por lo que no subo ninguna version web.

En ambos dos ficheros he dejado una dirección de email para que el que los desencripte me conteste.

Por cierto, agradecer a Carlos su conversion del Algoritmo a PHP. Puedes descargarte si quieres tambien el fuente. Aunque el cree que es posible, yo estoy convencido de lo contrario, asi que para el primero que me mande las 2 password de los ficheros encriptados (a la direccion contenida dentro de ellos) tendra la bonita recompensa de este dominio totalmente gratis para el solito !!!
